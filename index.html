GeoFarming

GeoFarming App: A smart solution for cattle paddock management using geospatial analysis. Built for large-scale farmers, it enables efficient paddock mapping, grazing optimization, and rotation planning.

Users can map paddocks using Google Maps (by marking or walking), input key parameters like grass cover and intake, and generate optimal divisions using Voronoi diagrams. The app streamlines grazing strategy, enhances productivity, and improves land utilization.

Key Features: Firebase user profiles, paddock mapping (mark/walk), division calculator, and wire placement guidance.

Tech Stack: Java, Firebase Realtime DB, Google Maps API
